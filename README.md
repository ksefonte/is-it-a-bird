# Is it a bird?

Or is it a plane. That's the question.

Crowdsourced investigative journalism, helping sky-gazers around the world answer one simple question - is it a bird, is it a plane?

## How does it bird?
The user spots an unidentified flying object in the sky - a potential bird, or PB - and pulls the app. Leveraging Open Access air traffic control data in conjunction with device sensor APIs for acceleromter, gyroscope, magnetometer to bind the phones viewing angle, we  determine whether or not the unidentified flying object is a plane.
